A modern, feature-rich personal finance tracker built with vanilla HTML, CSS, and JavaScript. Track income, expenses, visualize trends, and export your data — all in a beautiful, responsive interface.

## 🚀 Features

### Core
- ✅ Add, edit, and delete income & expense transactions
- ✅ Category-based tracking with emoji icons (8 expense + 5 income categories)
- ✅ Automatic balance, income, and expense calculation
- ✅ Data persistence using LocalStorage
- ✅ Form validation with inline error messages

### Visualization
- 📊 **Doughnut chart** — Expense breakdown by category
- 📈 **Bar chart** — Monthly income vs. expense trends (Chart.js)

### Search & Filter
- 🔍 Real-time search across descriptions, categories, and amounts
- 🏷️ Filter by type (Income / Expense) and category
- 🔃 Sort by date (newest/oldest) or amount (highest/lowest)

### Data Management
- 📥 **CSV Export** — Download all transactions as a CSV file
- 🗑️ Clear all transactions with confirmation
- ✏️ Edit transactions via modal dialog

### Design
- 🌗 Light / Dark mode with persistent theme preference
- 🎨 Glassmorphism UI with smooth animations and gradients
- 📱 Fully responsive layout (mobile, tablet, desktop)
- 🔔 Toast notifications for user feedback
- ♿ Accessible — ARIA labels, semantic HTML, keyboard support
